# XJQQ Bnl website download

Use this skill when the user wants to download the bnl files for all the XJQQ books in a picture.

## Workflow

1. Use the book-chinese-title-detection skill to detect all the book titles in the picture.
2. For each book title, use the xjqq-bnl-website-download skill to download the .bnl file.
